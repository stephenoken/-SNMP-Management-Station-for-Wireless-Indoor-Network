# -SNMP-Management-Station-for-Wireless-Indoor-Network

## Project Dependancies
- `Brew install libsmi`
- `Brew install net-snmp`

## Run example script
- `python src/pysnmp_example.py`
- `snmpwalk -v2c -c public 127.0.0.1 .1.3.6.1.2`
